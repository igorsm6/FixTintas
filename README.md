🎨 FixTintas
Sistema em desenvolvimento para gerenciamento de uma loja de tintas, criado com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO) em C#.
________________________________________
📚 Sobre o projeto
O projeto está sendo desenvolvido como trabalho acadêmico, com foco em:
•	Aplicação dos 4 pilares da POO:
o	Encapsulamento
o	Herança
o	Polimorfismo
o	Abstração
•	Organização de código em camadas
•	Estrutura preparada para integração futura com banco de dados e aplicações web
________________________________________
🧱 Estrutura do projeto
O projeto está organizado da seguinte forma:
📁 Models     → Classes que representam os dados (Cliente, Produto, etc)
📁 Services   → Regras de negócio e operações (CRUD)
📄 Program.cs → Ponto de entrada do sistema
________________________________________
⚙️ Funcionalidades atuais
✔ Cadastro de produtos (em memória)
✔ Estrutura inicial para cadastro de clientes
✔ Organização em Models e Services
✔ Simulação de operações CRUD
________________________________________
🚧 Em desenvolvimento
•	Cadastro completo de clientes
•	Cadastro de fornecedores
•	Integração com banco de dados
•	Validações mais avançadas
•	Interface (futuramente web)
________________________________________
💡 Tecnologias utilizadas
•	C#

•	Programação Orientada a Objetos

Preparar a base do sistema para evoluir futuramente para uma aplicação completa, seguindo boas práticas utilizadas no mercado.
